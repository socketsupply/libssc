ssc_init_argc(3) -- Socket SDK Client Library
====================================================

## SYNOPSIS

  `#include <ssc/ssc.h>`

  `const int ssc_init_argc();`

## DESCRIPTION

Returns the initialized argument count given to `ssc_init`() or
`ssc_init_library`().

## RETURN VALUE

The initialized argument count.
