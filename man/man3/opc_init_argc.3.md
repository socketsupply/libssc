opc_init_argc(3) -- Operator Framework Client Library
====================================================

## SYNOPSIS

  `#include <opc/opc.h>`

  `const int opc_init_argc();`

## DESCRIPTION

Returns the initialized argument count given to `opc_init`() or
`opc_init_library`().

## RETURN VALUE

The initialized argument count.
